Files uploading on Website

The modules allows to add downloads to your odoo websites.

Usage

To create new download switch to editor mode, put cursor and click Image button on the editor bar.

Upload new file or select one from existed

Alternatively, Link Editor can be used.

Backend

Uploaded from website files are marked at backend.
